# Frequently Asked Questions (FAQ)

## O que é o Java on IBM Z Simulator?

O **Java on IBM Z Simulator** é uma plataforma educacional gamificada desenvolvida para ajudar programadores COBOL e profissionais IBM Z a aprender Java de forma prática, utilizando laboratórios, quizzes, desafios, arquitetura interativa e sistema de progressão.

Faz parte do **Bellacosa Mainframe Learning Framework**.

------

# Quem pode utilizar?

O simulador foi desenvolvido para:

- Programadores COBOL
- Desenvolvedores Java
- Estudantes
- Profissionais IBM Z
- Arquitetos de Software
- Instrutores
- Empresas que treinam equipes IBM Z

Não é necessário conhecimento prévio em Java.

------

# O simulador é gratuito?

Sim.

O projeto foi criado com fins educacionais e pode ser utilizado para estudo e treinamento.

------

# Preciso instalar alguma coisa?

Não.

O simulador funciona diretamente no navegador.

Também pode ser instalado como **Progressive Web App (PWA)** em dispositivos compatíveis.

------

# Funciona offline?

Sim.

Depois do primeiro carregamento, os principais recursos ficam disponíveis offline graças ao Service Worker e ao cache da aplicação.

# Existem laboratórios práticos?

Sim.

O projeto inclui laboratórios guiados que simulam situações comuns encontradas em ambientes IBM Z.

Cada laboratório possui:

- objetivo;
- nível de dificuldade;
- exercícios;
- solução;
- XP.

------

# O que são Boss Battles?

São desafios finais que consolidam os conhecimentos adquiridos.

Ao derrotar um Boss, o aluno recebe XP e pode desbloquear novas conquistas.

------

# Como funciona o sistema de XP?

Cada atividade concede experiência.

Exemplos:

- conclusão de laboratórios;
- quizzes;
- desafios;
- Boss Battles;
- conquistas especiais.

Ao acumular XP, o jogador sobe de nível.

------

# O que são conquistas?

As conquistas recompensam objetivos específicos, como:

- concluir laboratórios;
- atingir determinada quantidade de XP;
- derrotar Bosses;
- gerar certificados;
- explorar funcionalidades do simulador.

------

# Posso gerar certificado?

Sim.

Ao concluir os requisitos definidos pelo curso, o simulador permite gerar um certificado personalizado.

------

# O certificado possui validade oficial?

O certificado demonstra a conclusão das atividades dentro do simulador.

Caso o projeto seja utilizado por uma instituição de ensino ou empresa, a política de certificação dependerá dessa organização.

# Como alterar o idioma?

Acesse **Configurações → Idioma** e escolha um dos idiomas disponíveis.

Atualmente:

- Português (Brasil)
- English

Novos idiomas poderão ser adicionados futuramente.

------

# Onde os dados são armazenados?

Por padrão, o progresso é salvo localmente no navegador utilizando **Local Storage**.

Versões futuras poderão oferecer sincronização em nuvem.

------

# Posso exportar meu progresso?

Sim.

O simulador permite exportar informações de progresso e estatísticas para facilitar backup e análise.

------

# Os dados pessoais são enviados para servidores?

Na versão atual, o simulador foi projetado para funcionar localmente. Caso recursos online sejam adicionados no futuro, a documentação correspondente detalhará o tratamento dos dados.
