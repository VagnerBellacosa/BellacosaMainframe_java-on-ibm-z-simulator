# Code of Conduct

## Nossa Comunidade

O **Bellacosa Mainframe Learning Framework** busca criar um ambiente acolhedor, respeitoso e colaborativo para todas as pessoas interessadas em aprender e compartilhar conhecimento sobre IBM Z, Mainframe e desenvolvimento de software.

Independentemente do nível de experiência, todos são bem-vindos para participar.

------

# Nosso Compromisso

Como participantes, colaboradores e mantenedores deste projeto, comprometemo-nos a promover uma comunidade baseada em:

- respeito;
- cordialidade;
- colaboração;
- inclusão;
- aprendizado contínuo;
- profissionalismo.

Nosso objetivo é incentivar um ambiente seguro para troca de conhecimento técnico.

------

# Comportamentos Esperados

Esperamos que todos os participantes:

- sejam educados;
- respeitem opiniões diferentes;
- aceitem críticas construtivas;
- compartilhem conhecimento;
- ajudem iniciantes;
- mantenham discussões técnicas produtivas;
- valorizem a diversidade de experiências.

------

# Comportamentos Inaceitáveis

Não serão tolerados:

- ataques pessoais;
- assédio;
- intimidação;
- discriminação;
- linguagem ofensiva;
- ameaças;
- divulgação de informações privadas sem autorização;
- spam;
- trolling;
- comportamento abusivo.

Também não são apropriadas discussões que desviem repetidamente o foco técnico do projeto.

------

# Discussões Técnicas

Debates técnicos são incentivados.

Discordâncias fazem parte do desenvolvimento de software, mas devem ocorrer com respeito e foco em argumentos técnicos.

------

# Revisões de Código

Durante revisões de Pull Requests:

- critique o código, não a pessoa;
- explique o motivo das sugestões;
- proponha melhorias sempre que possível;
- mantenha um tom profissional.

------

# Comunicação

Este projeto valoriza uma comunicação clara e respeitosa.

Ao responder dúvidas:

- seja paciente;
- considere diferentes níveis de experiência;
- incentive o aprendizado.

------

# Responsabilidades dos Mantenedores

Os mantenedores podem:

- solicitar ajustes em contribuições;
- editar documentação;
- fechar Issues ou Pull Requests fora do escopo;
- remover conteúdo inadequado;
- aplicar este Código de Conduta quando necessário.

As decisões serão tomadas buscando preservar um ambiente saudável para toda a comunidade.

------

# Consequências

Dependendo da gravidade, poderão ser aplicadas medidas como:

1. orientação informal;
2. advertência;
3. remoção de conteúdo;
4. bloqueio temporário;
5. bloqueio permanente de participação no projeto.

------

# Escopo

Este Código de Conduta aplica-se a:

- Issues;
- Pull Requests;
- Discussões;
- Documentação;
- Código-fonte;
- Materiais publicados neste repositório.

Também se aplica quando alguém representa oficialmente este projeto em eventos, comunidades ou redes sociais.

------

# Reportando Problemas

Caso encontre comportamento incompatível com este Código de Conduta:

- abra uma Issue privada, quando possível;
- ou entre em contato com os mantenedores do projeto.

Os relatos serão analisados com confidencialidade e seriedade.

------

# Nossa Filosofia

Acreditamos que o conhecimento cresce quando é compartilhado.

Este projeto existe para ajudar profissionais, estudantes e entusiastas a aprender tecnologias IBM Z por meio de conteúdo prático, acessível e colaborativo.

Toda contribuição positiva fortalece essa missão.

------

# Atribuição

Este documento foi inspirado no **Contributor Covenant 2.1**, amplamente utilizado em projetos de software livre, e adaptado às necessidades do **Bellacosa Mainframe Learning Framework**.
