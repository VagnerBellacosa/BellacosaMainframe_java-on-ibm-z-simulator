# Version 1.0.0

## Primeira versão pública

Esta versão marca o nascimento do **Java no IBM Z Simulator**, um laboratório gamificado desenvolvido para auxiliar programadores COBOL a compreenderem como Java atua dentro do ecossistema IBM Z.

O projeto foi concebido para funcionar inteiramente no navegador utilizando HTML5, CSS3 e JavaScript moderno.

# Problemas conhecidos

Nesta versão:

- Não há backend.
- O progresso é armazenado apenas no navegador.
- O certificado ainda não possui assinatura digital.
- O ranking é local.
- Os efeitos sonoros dependem das permissões do navegador.
- O funcionamento offline (PWA) ainda não está disponível.
